# Code of Conduct

This project is a place to build good software together.

**Expected:** be respectful and constructive; critique code and ideas, not
people; assume good faith; welcome newcomers and different levels of
experience; keep discussions on topic.

**Not tolerated:** harassment, personal attacks, discriminatory language,
sustained disruption, publishing others' private information, or any conduct
that would be unacceptable in a professional setting.

**Scope:** all project spaces — issues, pull requests, discussions, and any
official channels.

**Enforcement:** report conduct issues privately to
**tarun.vangari@gmail.com**. Maintainers may edit, remove, or reject
contributions and comments that violate this code, and may temporarily or
permanently ban contributors for repeated or severe violations. Reports are
handled confidentially.
